# Vitrine de Links (tipo Linktree) — Cabeçalho com faixa e avatar

Projeto estático (HTML/CSS/JS) com *Admin (login: senha admin)* para criar sua vitrine e *Vitrine* para exibir.

Cabeçalho no estilo do exemplo enviado: *faixa roxa, **avatar circular* centralizado com *borda branca, e **título/subtítulo* abaixo.

## Como usar

1. Abra admin.html e faça login com admin.
2. Em *Configurações, informe **Logo (URL), **Título, **Descrição, **Cores* e *Layout*.
3. Adicione seus itens (imagem por arquivo ou URL, título, descrição, link, CTA).
4. Clique em *Gerar link* para obter vitrine.html?data=... e compartilhe.

## Publicar no GitHub Pages
- Suba todos os arquivos para um repositório e habilite *Settings → Pages*.
- Acesse https://SEU_USUARIO.github.io/NOME_DO_REPO/admin.html para montar sua vitrine.
- Envie o link gerado vitrine.html?data=... para clientes.

## Observações
- Login apenas client-side (não protege dados sensíveis).
- Para links mais curtos, prefira *Imagem (URL)* em vez de upload de arquivo (data URL).
